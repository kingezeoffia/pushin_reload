# Custom Workout Icons

Add your custom workout icons here as PNG files:

- `pushups.png` - Push-ups icon (56x56 recommended)
- `squats.png` - Squats icon (56x56 recommended)
- `glute_bridge.png` - Glute Bridge icon (56x56 recommended)
- `plank.png` - Plank icon (56x56 recommended)
- `jumping_jacks.png` - Jumping Jacks icon (56x56 recommended)

## Icon Guidelines:
- **Size**: 56x56 pixels or larger (will be scaled down)
- **Format**: PNG with transparent background
- **Style**: Clean, simple, Apple-inspired design
- **Color**: White or transparent (color will be applied programmatically)

## Current Fallback:
If an icon file is missing, the app will show a default fitness icon as fallback.
