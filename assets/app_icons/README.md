# App Icons for Most Used Apps Widget

Place PNG files of app icons in this directory with the following naming convention:

- `instagram.png`
- `youtube.png`
- `tiktok.png`
- `twitter.png`
- `facebook.png`
- `snapchat.png`
- `whatsapp.png`
- `reddit.png`
- `netflix.png`
- `spotify.png`
- `game.png`
- `safari.png`
- `chrome.png`

## Icon Specifications:
- **Format**: PNG with transparent background
- **Size**: 64x64px or higher (will be scaled down)
- **Style**: App store icons or clean app icons
- **Naming**: All lowercase, no special characters

## Fallback:
If an app icon is not found, the widget will automatically fall back to using Material Design icons.

## How to get app icons:
1. Search for "[app name] icon PNG transparent" on Google Images
2. Download clean, square icons
3. Resize to 64x64px or larger if needed
4. Save with the exact filename listed above

The widget will automatically detect and use these icons when available!
